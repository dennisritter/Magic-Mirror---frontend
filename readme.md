# Perna App

Perna App ist eine skaliebare Implementierung eines smarten Spiegels, die es dem Nutzer eine individualisierbare Oberfl√§che bietet. Perna App bietet zusammen mit Perna API ein Plattform zum Betrieb von derartigen Spiegeln. 


## Installation
#### Entwicklungsumgebung
    - npm install
    - bower install
    - gulp build
    - gulp ( BrowserSync Server an localhost:3000 )

#### Distribution
    - Deploy des dist-Ordners in eine Serverumgebung
